使用印象笔记的一些收获

发表于 2016-12-26 | | 阅读次数

首先，我是一开始在行动派琦琦的推荐下知道了这么一个好的笔记平台，印象笔记给自己的宣传语为“你的第二大脑”，刚刚开始也是对于印象笔记没有什么感觉，甚至都不太会用这么一个好东西，慢慢地在摸索中逐渐发现原来我已经离不开它了。

不过，现在好像我没有太大的头绪来写这篇文章，其实说实话，道理都懂，也不笨，收藏了很多资料，也整理了很多次，但是就感觉没有发挥太大的作用一样。
其实，印象笔记作为笔记方面的老大，很多方面都已经做得非常成熟，同时各种教程也都非常多，数不胜数，只要看一两个就足够你好好学习探究这个笔记软件了。

那么，作为学生的我们，其实大部分人都没有做笔记的习惯，甚至还停留在纸质版笔记的时代，我并不是说这个方式不好，相反，我认为纸质版的笔记是云笔记永远无法取代的。比如说：“当你在听课的时候，你只能通过笔来把知识重点圈在课本或者记下来，或者当你在看mooc课程的时候，你的电脑可能不支持视频与笔记共同在一个桌面上，这个时候，你需要的最好的办法就是手写，把重点记录下来，待课程结束后再来慢慢整理知识点。同时，手写笔记能够帮你理清思路，进一步分析你所学的东西，手脑并用，这样你也才能印象深刻。

所以，尽可能地使用你的笔吧，能多挥洒就多挥洒。

现在的话，我是云笔记与纸质笔记一起协同使用，这能够帮助我提升一些效率，使我能够有一些精力来做其他事。

接下来，我会按照李笑来老师的所说的那样，教是最好的学，同时写作也能够检验你到底对这东西懂不懂，来说说我使用印象笔记以来的一些收获。首先，我们要清楚一个观念，在互联网时代，你的一切信息都几乎能够在网上搜索得到，一切知识也能够在网络上搜索得到，我们就生活在云时代里，所以我们也有必要把我们的资料，信息放在一个云端的全平台上。

注意：这样的平台一定要是符合三大特点：

  • 全平台，包括主流的Windows，Mac，iOS，Android等等
  • 一定要能够确保你的资料能够保存下来，有强硬的靠山，比如百度云，360网盘等等，不过目前我最信赖谷歌云盘，只不过被墙得太厉害，经常需要科学上网，搞得很累
  • 能够保证你的数据是安全的

所以，综合来看，目前你可以选择印象笔记这样一款全平台的产品，值得信赖，就算这个公司倒闭了，你的数据依然也能够保存在本地（电脑的某个盘上）。来，上个图。

导入导出文件，非常方便，妈妈再也不担心我会丢失文件了。当然，你得付费才能加密文件，不过一般对于我这种日常使用者来说，一般没有太重要的东西需要加密，况且很重要的东西你也不可能放在云上的嘛！

我身边的一些朋友虽然也使用云，不过一般是百度云等，360云盘等，不过这些云盘只是用来存储文件的，不适合用来存放资料，因为这两个本就是不同的概念。当然，我也见到一些朋友，比如手机丢了的，刷机了的，换手机的，然后各种关于数据的问题就来了。朋友圈各种喊：“亲们，手机丢了，麻烦把手机号再发一下”，或者刷机了，重要的数据也丢了，对此，我当然是一脸嫌弃。。。因为他们没有使用该有的工具，何必再麻烦别人呢？

因为所有的这些东西都可以依靠云存储来解决，就这样。

那么，接下来，我就自己使用印象笔记的一些技巧，感悟来做一些介绍。主要是三大模块。

  1. 使用印象笔记的一些技巧
  2. 如何利用印象笔记来做一些知识性的积累
  3. 使用印象笔记，我的一些收获

1.使用技巧

1.1 构建你的第二大脑

1.1.1 用好标签神器

印象笔记目前支持2级笔记本，标签也是多的足够你用。并且标签支持多层级叠加，无限放置。打个比方，我们传统的笔记本架构是多个层级笔记本，这样一层一层分类下去，但是当你有这么一个文件具有多重属性的时候你会把它放在哪个文件夹下，比如，中国杭州G20峰会的有关资料与新闻，它可能与国家，政治，经济，还可能与互联网有关系，那么这时候你会把这个资料放在哪个电脑文件夹下呢？国家？经济？或者两个文件夹都放？

很显然这种分类方式已经不足以对应我们这个大数据时代了，而标签却很好解决这个问题，一个笔记可以对应N个标签，只要它有相应的属性，对上面那个比方刚刚好合适。A的属性有B,C,D；同时当你需要这个笔记的时候，随便输入一个关键词标签都可以找得到这个笔记，非常强大。再如下面这个笔记的标签对应的是微信，北京化工大学。

1.1.2 构建有序的笔记本列表

由于印象笔记只有两个层级的笔记本，刚开始用的时候，我也是有点不习惯，觉得印象笔记的产品经理绝对是傻，才两个层级，怎么够用嘛。然而真相是足够了。
那么如何给笔记本命名呢？
我一般的做法便是以A,B,C来开头命名，下一个层级再是A-01，A-02，B-01，B-02。
为什么前面要加个0呢，这是因为一般笔记数量都会有数十个，如果多的话甚至可以设置001这样，或者以 I,II,III,这样来命名也行，或者1,2,3都可以，这样你的笔记本就会排序非常整齐，这是一般的命名规则，对于某些日记类的可以以日期开头，如“2016.9.23+印象笔记”这样来命名，也是可以的，总之，发挥你们的想象力去做吧。这些命名规则同样适用于电脑文件命名，网盘文件夹命名等等。

1.1.3 强大的搜索功能

印象笔记有非常强大的搜索功能，不仅支持全文搜索，即在全笔记本中搜索你输入的关键字，还可以搜索图片上的文字，支持OCR扫描搜索，搜索出来的结果都会有黄色的框框给框起来。比如我想找有关罗辑思维的一些笔记。

1.2 知识性积累玩法

1. 如何使用印象笔记微信，微博，知乎等插件来保存信息

首先在微信上可以关注我的印象笔记，这个公众号来保存你想要看的微信文章，比如你平时需要学习哪些知识，急需哪些资料等等都可以一键发送到印象笔记，微博则是关注@我的印象笔记，以后只要你觉得哪个知识你需要都可以通过@我的印象笔记来保存这条笔记，所有这些笔记都是永久地保存在你的印象笔记上面的，如果它完整地剪藏下来的话。

比如，微信平台的某篇热文被你剪藏了，而过后原作者把它删除了的话，你依然还拥有这条笔记，因为它就在你的印象笔记里，一直都在，除非你自己手动删除。
具体的操作方法我就不介绍了，自己去摸索或者网上找教程。

看到没有，效率杠杠的。接下来还有一个大招。

2. 使用IFTTT让你的印象笔记与整个互联网相连

因为印象笔记开放的接口比较多，所以未来第三方的程序会越来越多地接入到印象笔记中，让你享受一把知识的快感。
那么，IFTTT是个什么玩意呢？它是一个可以让你在A软件做的事能够同步到B软件上，保持同步，或者你也可以通过A到B到C这样，不怕你到不了，就怕你想象力不够，当然，这也的取决于其他软件是否开放接口了。不止可以同步到印象笔记里，还可以同步到其他APP了哦。
比如我在Instagram发了一张图片，我想当我发出这张图片的时候能够同步到我的印象笔记里面，而我又不需要做其他的操作，只需要在IFTTT上面提前把你的Instagram账号与你的印象笔记账号连接起来，到时候就可以自动同步啦。

IFTTT全平台支持，手机即可操作，简单方便。让你的学习，生活都更加方便。

3. 电脑上的印象笔记剪藏功能

如果说一个APP能做的这么优秀已经很不错了，然而这个功能我认为真的非常强大，你可以选择任意方式来剪藏电脑上的网页。

  1. 网页正文
  2. 隐藏广告
  3. 网址
  4. 整个页面
  5. 屏幕截图

隐藏广告这个功能真的太好了，只剪藏我需要的东西，而我不需要的东西全部pass掉，这种清爽的感觉真美好。

如果还没用过赶紧用起来，对了，这个插件支持chrome，360浏览器，猎豹浏览器，还有UC浏览器哦，其他浏览器我还没试过，不过一般也就用这几个浏览器吧，反正我已经舍弃其他浏览器，用chrome了。

印象笔记还有一个大功能，那就是手机端拍照支持将照片以白板样式的文档格式保存，亦即去除杂质，让你的文档更加清爽。

2. 知识的积累，信息的保存

知识的积累是一点一点来的，平时微信看到好的文章了，微博看到好的资料了都可以一键保存到印象笔记里面。

不过，我们要有意识这样去做，也就是提前知道自己想要了解什么，想要学习什么，或者又想了解哪个概念，提前订好自己的收集目标，免得看见什么都想收集。
收藏知识的法则有几个：

  • 网络上能够经常找得到的知识不用收藏，比如百度百科，又或者是某个概念的定义，大可不必收藏；
  • 新闻性，时效性非常高的文章也不需要收藏；
  • 与自己的目标无关的知识也不需要收藏；

因为时效性高的文章很快就会随着时间慢慢地就没有价值了，比如某些新闻，某些娱乐类的信息，当然如果自己就是专门研究这个的另外说。而与自己目标不相关的文章即使再有意义，再重要也不要把它放到你的笔记本里，因为那会扰乱你的知识体系，甚至让你开始浮躁起来，只有足够专注，才能够在这个世界走出属于自己的一片天空。

需要收集大量的文章的话推荐一个APP，pocket（稍后阅读类），相关话题以后再说，等我慢慢用熟之后再说。

有了收集，那么接下来一步就是要整理了，整理是必不可少的，否则，就是收集了那么多，你依然还是没有智慧。

信息永远不能转化为智慧，只有当经过人的思考，整理后才能变得有价值。否则，那百度，谷歌等搜索引擎早就变成智人了。
整理也有三个原则：

  • 过时的删掉
  • 当时看起来非常有意义的然而现在这块知识已经内化为你的经验的时候，果断删掉
  • 将来也不可能用的上的笔记，删掉。

这些笔记当然不包括自己的日记类留念记录笔记，而是关于自己成长，认知方面的笔记。试试这三个原则，我相信应该会有不错的进步。来，上图。

3. 使用印象笔记的一些收获

其实，也无所谓收获不收获，只是一个工具罢了，工具的诞生本来就是为了让我们的生活更加美好，然而我有一段时间却在纠结于该使用哪个工具好，关于笔记平台，我几乎尝试了市面上所有的平台，为知笔记，有道云笔记，最终还是印象笔记比较适合我，因为已经离不开了。

其实选择也是一种进步，因为选择，你肯定是要选择最适合自己的，自己最习惯的那个东西，我想这应该同样适用于我们的生活。一开始纠结，后来慢慢找到适合的，再后来各取优点，不再为它的缺点，它没有的功能而懊悔，毕竟你的选择多的是，一旦认定，习惯了一个适合自己的东西，再也回不到以前了，所以我们要学会拥抱变化。就像那些一直还徘徊在office2003的人一样，2016版本再好，因为不习惯，那也没有办法。

所幸，我还将一直在路上，尽管有时候还是会很无知，但一直在努力让自己更美好。
