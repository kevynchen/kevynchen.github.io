我的一些读书经历

发表于 2017-04-23 | 分类于 生活

我的一些读书经历

这几天，在读几本书，大数据时代，王小波三部曲，还有浪潮之巅，感觉读书的时间过得非常地充实，比刷所谓的朋友圈或者微信文章有意思的多了。

初中阅读史

回顾我的读书历史，大概是从初中的时候开始正式的阅读的吧，那时候整个镇上只有一个书店，并且里面全部都是盗版书，而且还是我们初中学校的一个老师开的，我记得我的第一本启蒙读物是圣经吧，那是初一的时候，每周放一次假，只能趁着周日去上学的时候，路过那个书店，于是在拥挤又有些昏暗的书店里一点点地阅读那一本《圣经》，读了好几个星期还是没有读完，不过，里面的剧情当时还是十分地吸引我，“人类”，“上帝”，“诺亚方舟”，“亚当和夏娃”，当时觉得竟然还有这么一方世界，觉得奇妙极了。不过，也不知道怎么地，那本书至今都没有读完过。

再长大一些，在课文里学到了“保尔柯察金”，就是那个《钢铁是怎样炼成的》里面的主人公，刚好我的同学拥有这样一本书，等他读完了之后，我便借来看了一下。不过，好像一开始并没有真正阅读，而是在一个百无聊赖的下午，才想起有这么一件事可以做，在家里也不太适合读书，于是溜到了一个经常晒萝卜的地方去读这一本书，然后就一发不可收拾，具体的细节已经记不清了。甚至保尔这个人也记不清了，不过，阅读的经历却保留了下来。

当时，学校初一校区仅有一个限时的阅读室，每天下午5点半–6点半开门，这之后就关门了。那时偶尔也去那里读一读文摘，不过半个小时还是挺短的，毕竟要吃饭，还得冲凉。

再然后，就接触了一些路遥的作品，《人生》这本书留给我的印象很深刻，又一个第一次开始真正意义上的接触了好的作品，那时候一个好同学还为此模仿了路遥的人生这部小小说，我还有幸看了他写的一小段，觉得还是不错的。

然后，镇上那个书店也租出去了，又变成了一个出租各种小说的店，“言情小说”，“魔幻小说”等等，那时候，我还读过一些诸如《坏蛋是怎样 炼成的》等等的书，现在我都还记得深夜12点拿着厚厚的《哈利波特7部合集》进行阅读的那样一种感觉，既好奇又害怕，大脑里整个地都在想像这哈利波特是如何骑着他的魔幻扫帚，伏地魔又是如何把他的父母打死的等等，对了，还有经典的“麻瓜”…..

再后来，镇上一个书店也没有了。

初中时代的阅读不多，也许加起来所有的书都不超过20本，但那肯定是有重量的，至少我开始会选择哪些书值得一读，哪些书不值得一读，以前总觉得那些文采好的散文简直就是好文章，而现在看来，也许是矫情过度，什么都不是了吧。还有魔幻之类的，言情之类的小说也绝对不会再阅读了。

后面的阅读经历就更加清晰了一些，毕竟高中时代开始了。

高中阅读史

很庆幸，高中校门外，每次周日，那个山东大哥总是在那里摆摊，那里有各种各样的杂志，《读者》，《兵器世界》，《意林》，《故事会》，甚至还有《知音》 等等，还有各种各样的小说，每一次我都能够在那里淘到不少好书，甚至还能淘到一些正版书，正版书是多么地珍贵，举一个例子，当你在写数学题的时候，你去查答案，很明显，你写的是对的，然而却怎么也对不上答案，最后却发现越来是题目漏了某一个东西，不过，现在复印的技术应该变得好多了，正版与盗版没什么差别了吧，只是所用的纸张有所差别而已。

那时候，县里只有两个书店，新华书店和百川书店，闲着没事的时候，也会到书店里看看书，只是碍于正版书价格太贵，总是不忍心去花几十块钱。算了，就这样罢，于是每次都是读到一半，又扔下，直接回学校了。

印象深刻的书也不多，平凡的世界，世界上最伟大的推销员，其他杂七杂八的小说也没什么印象了。具体给我带来深刻影响的是那一本步行的书，后来我就萌生了有一天也要去走走的愿望。

读书很杂，那是因为在探索阶段。那时候，我总是觉得应该多读一些书，因为上大学了，工作了，你根本就没有时间读书了，或者说你根本就不会再读书了，当然我那时候也是这么对我的同桌说的，我也是这么做的，可是现在来看，那时候的感觉真是个错误，大学的时间多得是，只要你想读，你就可以阅读，只是你得真正地去执行下去。

当然，很悲哀，现在我的阅读也渐渐地少了些，也开始了有目的的去阅读，这也许就是一种成长吧。

现在，虽手里有kindle，学校有图书馆，有很多的资源，但都没有好好地利用，浪费了。仔细想想看，知识从来都在那里，可是我们为什么都不愿意去翻开呢，一方面，我觉得是互联网让我们的阅读能力下降了，再也没有那么多耐心去阅读了，另一方面，互联网的信息流实在是太过于强大，乃至于我们无法真正地抽出一个完整的时间去阅读，因为网上有趣的事情太多了，我们很难克制住自己。因为这些信息流刚好与人对这个世界很好奇这个人性相符合，所以也就足为怪了。

焦虑，烦躁，不知所措，这些都不应该我们不阅读的理由，毕竟当我们真的沉浸在阅读当中时，那份感觉太妙了。不可否认，我们阅读，最终是为了成为一个最好的自己，如果现在就有这样的一种技术，直接将人类的知识能够直接输入你的大脑，让你不用再阅读任何东西，甚至是不用学习，可想而知，我们再也不用受教育了，事实上，这样的事情正在发生，很多机器人可以轻而易举地超越我们的大部分普通人的知识容量，很多工作就这样被机器人取代了。

世界读书日，不仅仅只是今天读书，更是要每天都抽出一些时间来阅读。

也许，以后，我们见面的第一句话便是“你今天读了吗？”。
