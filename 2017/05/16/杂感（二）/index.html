杂感（二）

发表于 2017-05-16

Ubuntu系统折腾

星期天的时候看到新闻上面说微软将要在其应用商店里面提高Ubuntu等操作系统的安装运行包，于是就有点动心了，毕竟对于开发人员来说，linux是一个比较理想的开发环境，然后，我就装了个虚拟机，装了一个Ubuntu的桌面操作系统，不得不说，对于新手来说，非常地不友好，基本都是用命令行操作的，真的是太困难了。

很多日常用的软件系统上面也没有，这让我很头疼，我连解压文件，怎么翻墙，安装一个软件都还弄不太懂，于是就在虚拟机里折腾了两天，当然，净是折腾Mac主题的桌面了，然后我发现Mac系统原来也不过如此，体验过了一把新鲜劲后，就没有任何的心意了，甚至想要回原来简洁的操作系统，稳定，安全，也许也不会占用太多的资源。

所以在折腾完了Mac主题之后，我索性把那些花哨的东西给全部删掉了，只留下了最基本的Mac主题的一些东西，不过不得不说，Ubuntu的terminal真的太好用了，并且颜值也非常地高，比Windows上面的cmd好地太多，简直就不是一个世界的东西啊。

不得不说，如果想要专注的高效开发，桌面系统真的就应该抛弃掉，只留下命令行式的操作，不过，我目前还做不到，很多东西都不懂，所以，昨天晚上挑选了一下Linux的入门书籍，在亚马逊下单了，估计今天就可以拿到这本书，大致了解一下Linux的系统。

骑行团队聚会

昨天晚上，我们5个人的骑行团队，第一次聚会吧，好像。大家的时间都很宝贵，难得有这么一块时间小聚一下，我真的有些怀念我们那些骑行的日子了，那时候什么也不用想，不用烦恼，一心向前即可。

而现在，他们都考研了，谈论的也都是考研的东西，是啊，不谈论这些东西，谈论点别的是不是有些浪费时间呢？看着他们好像考上了研究生，就好像是走上了一条美好的道路，我内心里替他们高兴。但是我也有些黯然神伤，因为我不考研，好像我的人生就这样了罢。年轻，有房，有车，谁不想要呢？但这好像不是我想要的东西，好像不是，对于这些东西，我好像并没有多大的兴奋，我是不是有些不正常。再这样下去，估计都平庸无奇了吧。

算了，想不了那摩多，暂且先这样，成功又如何，失败又如何。我只想让我的家人过上好日子。

就这样，无它。还有，实现自己的自由梦想，首先得事财务自由。
